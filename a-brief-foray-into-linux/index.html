A brief foray into Linux (2023)

Every few years I get the bug to try out Linux - is this the year of the Linux desktop? My very brief notes from the 2023 version are below. 

  • GhostBSD: buggy.
  • FreeBSD full version: install directory too small.
  • FreeBSD net install: fine, dull.
  • Debian stable: probably fine, don't like Gnome 3.38. Old packages in general.
  • Debian testing: fine. With various UIs:
    • Gnome 43: don't love.
    • Cinammon: ugly version.
    • KDE: ugly.
    • XFCE: fine, bit dull.
    • Mate: dull.
  • MX Linux: no, too lite.
  • Manjaro: fine.
  • Mint: good so far.

Addendum (March 2023): Mint was my favorite of the distros tested, by a surprisingly large margin. There is no single reason why - it simply felt like the best combination of reliability, features and polish. I did not stick with it, for Linux rather than Mint reasons - returned to Windows then to Mac.
